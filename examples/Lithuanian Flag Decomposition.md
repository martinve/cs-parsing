# Lithuanian Flag Decomposition

```
# ::snt The colors of Lithuanian flag are yellow, green and red.
(c / color-01
      :ARG1 (f / flag
            :mod (c2 / country
                  :name (n / name
                        :op1 "Lithuania")))
      :ARG2 (a / and
            :op1 (y / yellow)
            :op2 (g / green)
            :op3 (r / red)))
```
